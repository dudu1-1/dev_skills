## 我最喜欢的一段代码
```python
# dev_skills_env.py 核心代码：矩阵运算
import numpy as np

# 创建两个二维矩阵
a = np.array([[1, 2], [3, 4]])
b = np.array([[5, 6], [7, 8]])

# 矩阵加法
c = a + b
# 矩阵乘法
d = np.dot(a, b)

# 打印运算结果
print("矩阵加法结果：\n", c)
print("矩阵乘法结果：\n", d)
```

执行 `np.dot(a, b)` 可实现两个二维矩阵的乘法运算，执行 `print()` 语句可输出最终计算结果。
